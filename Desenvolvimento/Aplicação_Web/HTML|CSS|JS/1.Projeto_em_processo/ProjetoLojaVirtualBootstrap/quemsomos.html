Quem Somos

Quitanda Inline é uma loja virtual (e-commerce) fictícia criada como estudo de caso para o curso "Desenvolvimento de uma Loja Virtual com Bootstrap 5 e ASP.NET 5" disponibilizado gratuitamente pelo Profossor e autor Ricardo Maroquio em seu canal no YouTube.

O código-fonte pode ser obtido via GIHub, conforme orientações apresentadas ao longo dos vídeos do curso no YouTube. Este estudo e caso pode ser utilizado livremente como base para outras aplicações Web, sem ônus para o usuário.

Bons estudos e boa programação!
